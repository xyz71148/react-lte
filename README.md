# react-lte

=====


### 概述

[AdminLTE3](https://github.com/ColorlibHQ/AdminLTE) 的轻量级 react 封装。

### 开发

安装

```shell
git clone https://github.com/xyz71148/react-lte.git
cd react-lte
npm install
npm start
```

编译

```shell
npm run build
```
