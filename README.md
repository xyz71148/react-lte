### 开发

安装

```shell
git clone https://github.com/xyz71148/react-lte.git
cd react-lte
npm install
npm start
```

编译

```shell
npm run build
```

### cordova

    cordova create tutorial com.example.tutorial Tutorial
    cp tutorial/config.xml ./
    rm -rf tutorial
                    
    cordova platform add ios
    cordova platform add android
    cordova run ios
    cordova run android
